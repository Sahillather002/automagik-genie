## Key Concepts for Voice Agents
- **Transfer Protocol**: AG-UI provides a streaming protocol for agent communication
- **Mid-run Events**: Activity events allow tracking progress during execution
- **Session Logs**: Meta events and serialization enable session reconstruction
- **Interrupt Handling**: Built-in support for conversation interruptions
- **Event Streaming**: Real-time bidirectional event streaming via WebSocket

## Integration Points
- WebSocket endpoints for real-time streaming
- RESTful APIs for agent management
- Event-driven architecture for progress tracking
- Serialization protocol for session persistence
- Interrupt-aware lifecycle for voice interactions
